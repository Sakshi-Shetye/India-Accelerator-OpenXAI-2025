# MindMate - OpenxAI Global AI Accelerator Submission

## Overview
MindMate is a Next.js + Ollama AI project that provides personalized mental health support through AI chat. Users can get daily tips, guided exercises, and AI-powered responses to improve wellbeing.

## Submission Requirements

### Folder Structure
- **0001_MindMate/** contains the full Next.js project and supporting files.

### Project Information
- **Name**: Sakshi Shetye
- **Team Name**: Solo / MindMate Team
- **Project Name**: MindMate
- **Project Description**: AI-driven mental wellness assistant built with Next.js and Ollama AI for real-time personalized guidance.
- **Track**: Global AI Accelerator - Application Track

## Getting Started
1. Navigate to the `mindmate` folder.
2. Install dependencies: `npm install`
3. Run the app: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) to view in browser.
